
# 💰 SpendWise

SpendWise is a modern and intuitive expense tracking application built with Flutter. It helps users manage their daily finances, track spending habits, and gain insights into their financial activity.

## 🚀 Features

- 💵 Add, edit, and delete transactions
- 📊 Visualize spending with charts and analytics
- 🏷 Categorize expenses (Food, Travel, Bills, etc.)
- 📅 Track daily, weekly, and monthly expenses
- 🔍 Search and filter transactions
- ❤️ Clean and user-friendly UI
- ☁️ (Upcoming) Cloud sync and backup

## 🛠 Tech Stack

- Flutter
- Dart
- (Optional) Firebase / REST APIs
- State Management (Bloc / Provider / Riverpod)

## 🔄 CI/CD

This project uses GitHub Actions for:
- Automated build and testing
- Code analysis
- APK generation
